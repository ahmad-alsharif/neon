Job Market Analysis

SerpApi Pandas Matplotlib Seaborn

In this project, I used Python, SerpApi, and the Google Jobs Search Engine to collect and analyze 72 job postings for Data Analyst roles in Calgary, AB. My objective was to extract insights on the most in-demand technical and soft skills using natural language processing and visualization techniques.

֎ Key Takeaways:

    » Employers overwhelmingly expect Excel and SQL, with high demand for Power BI and Git as well

    » Job listings with higher skill scores tend to blend BI tools with coding and cloud platforms.

    » Strong skill co-occurrence patterns include:

  • - SQL + Excel: 22 jobs

    - Excel + Power BI: 17 jobs

    - SQL + Power BI: 16 jobs

    - SQL + Data Visualization: 16 jobs

    - Git + SQL: 13 jobs

    - Git + Excel: 13 jobs

    - SQL + Python: 11 jobs
